## wsl 
 WSL (Windows Subsystem for Linux) lets you run a Linux environment directly on Windows, without needing a virtual machine.
 This is very useful for tools like Foundry, Hardhat, Node, Git, etc., because many blockchain tools are built with Linux in mind.
 Think of it as a Linux terminal inside your Windows PC.
